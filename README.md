# HubSpot Docs Scraper

A Python TUI to scrape and store HubSpot API documentation as local Markdown files.

## Setup

```bash
pip install -r requirements.txt
python main.py
```

## Features

- 📋 Interactive TUI menu
- 🔍 Discover all HubSpot developer docs via sitemap
- 📥 Download only missing pages
- 🔄 Check for updated documentation
- 📁 Logical folder structure mirroring the docs URL hierarchy
- 💻 All code blocks preserved in Markdown
